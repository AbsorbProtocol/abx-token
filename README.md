# ABX Token

The native asset of ABSORB Protocol.

## Vision

ABX is the first token issued through Proof of Recovered Value (PRV).

Every ABX represents recovered value from digital assets that would otherwise remain unused.

## About ABSORB

ABSORB Protocol is building the Recovery Layer of Web3.

The protocol enables the recovery, consolidation and monetization of abandoned digital assets.

## Core Principles

- Proof of Recovered Value (PRV)

- Reserve-Backed Economy

- Transparent Recovery Registry

- Sustainable Token Issuance

- Community Governance
